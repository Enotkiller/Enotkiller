### 📌 Проекты

**[telegram-bot-for-college](https://github.com/Enotkiller/telegram-bot-for-college)**
Telegram-бот для колледжа с неудобным чередующимся расписанием (числитель/знаменатель).
Бот сам вычисляет тип текущей недели и присылает расписание перед парой.

- `/para` — какая пара сейчас (или что пар нет), можно уточнить конкретную пару по номеру
- `/para all` / `next` — все пары на сегодня / на следующий день
- `/cancel` — отменить конкретную пару на сегодня (не придёт уведомление)
- `/pingme`, `/pingwho` — подписка на пинг перед парой и список подписанных
- Данные — в Redis, деплой — через docker-compose
